# 极简智能退格逻辑

## 核心逻辑（超简化）

### 直接删除逻辑
- **当前行是空白行**：按退格键直接删除这一行
- **前一行是空白行**：回到前一行的缩进开始位置
- **前一行不是空白行**：回到前一行的末尾
- **非空白行**：使用默认退格行为

### 优势
- **无需双击**：一次退格直接删除空白行
- **简单直观**：用户容易理解和预测
- **保持结构**：智能的光标定位
- **易于维护**：代码逻辑极简

## 测试场景

### 场景1：空白行前是代码行
```javascript
method() {
    const x = 1;
    // ← 按退格：直接删除，光标到上一行末尾
    console.log(x);
}
```

### 场景2：连续空白行
```javascript
method() {
    // ← 按退格：直接删除，回到第4个空格位置
    
    // ← 按退格：直接删除，回到第4个空格位置
    
    console.log("test");
}
```

### 场景3：多层级代码
```javascript
method() {
    const y = 2;
        // ← 按退格：直接删除，光标到上一行末尾
        console.log(y);
}
```
